# Lab 00 : Configuration de l'environnement de lab

## Scénario du lab

Vous devez créer un nouvel abonnement Azure et demander une licence Azure AD P2 afin d'être prêt à réaliser les labs à venir.

#### Temps estimé : 5 minutes

## Créer un compte Azure et ajouter une licence Azure Active Directory Premium P2

Les tâches de cet exercice et les exercices de ce parcours d'apprentissage nécessitent que vous disposiez déjà d'un abonnement Azure utilisable. Si vous possédez déjà votre propre abonnement Azure, vous pouvez ignorer cette tâche et continuer à la suivante.

1. Dans un navigateur web, allez sur [https://azure.microsoft.com/fr-fr/pricing/purchase-options/azure-account](https://azure.microsoft.com/fr-fr/pricing/purchase-options/azure-account)

1. Faites défiler la page pour en savoir plus sur les avantages et les services  disponibles.

1. Sélectionnez **Démarrer gratuitement**.

1. Suivez l'assistant pour vous inscrire à votre abonnement Azure.

1. Vous aurez besoin d'une licence Azure AD P2 pour réaliser certains exercices. Dans l'organisation que vous avez créée, recherchez puis sélectionnez **Azure Active Directory**.

1. Dans le menu de navigation de gauche, sélectionnez **Commencer**.

1. Sous Commencer avec Azure AD, sélectionnez **Obtenir un essai gratuit pour Azure AD Premium**.

1. Dans le panneau Activer, sous **AZURE AD PREMIUM P2**, sélectionnez **Essai gratuit** puis sélectionnez **Activer**.

1. Dans le menu de navigation à gauche, sélectionnez **Aperçu**.

1. Actualisez le navigateur jusqu'à ce que vous voyiez Azure AD Premium P2 sous le nom de l'organisation. Cela peut prendre quelques minutes.

1. Vous devrez peut‑être vous déconnecter puis vous reconnecter à Microsoft Azure si vous rencontrez des problèmes avec des fonctionnalités attendues qui ne sont pas disponibles.
